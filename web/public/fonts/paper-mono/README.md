Paper Mono webfonts

Place the WOFF2 files for Paper Mono here. Expected filenames (adjust if your build produces different names):

- PaperMono-Regular.woff2
- PaperMono-Medium.woff2
- PaperMono-Bold.woff2

These are referenced in `web/src/app/globals.css` @font-face rules. If filenames differ, update the URLs there accordingly.

Source repository: <https://github.com/paper-design/paper-mono>
