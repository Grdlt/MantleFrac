First, run the development server:

```bash

# or
yarn dev
# or
pnpm dev
# or
```

Open [http://localhost:3001](http://localhost:3001) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.

## Environment Variables

### Required for Admin Features

- `ADMIN_API_KEY` - Server-only environment variable (not `NEXT_PUBLIC_*`) used to authenticate admin GraphQL mutations. Must match the `ADMIN_API_KEY` configured in the API service. Admin features like treasury setup, fee scheduling, and share supply configuration require this to be set.

### Other Environment Variables

- `NEXT_PUBLIC_API_URL` - GraphQL API endpoint (default: `http://localhost:4000/graphql`)
- `NEXT_PUBLIC_API_BASE` - Base API URL for proxy routes (default: `http://localhost:4000`)
- `NEXT_PUBLIC_NETWORK` - Flow network (default: `emulator`)
